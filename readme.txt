Poking Things With Sticks addons plugin.

A grab-bag of code to support the various non-standard things I do on my Poking Things With Sticks blog.


Build with "npx webpack".


Uses the following packages:

Dan Coulter's phpFlickr PHP wrapper for the Flickr API.  https://github.com/dan-coulter/phpflickr
Leaflet
jQuery.Sonar
DidactGothic free font
ChartJs
